### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ npm install -d
$ npm run start:dev
```

For production environments...

```sh
$ npm install --production
$ npm start
```
### Environments in production

- NODE_ENV
- DB_HOST
- DB_NAME
- DB_USER
- DB_PASSWORD
- ENCRYPTION_KEY
